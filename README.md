# [CNSL.LG-PROFILES]
### Minimalist Terminal-Themed Digital Presence

A clean, high-performance vertical landing page designed with a "terminal" aesthetic. This project serves as a focused portfolio representing the intersection of web development and data processing.

---

## // IDENTITY
- **Developer:** Evan Alifian
- **Focus:** Web Development & Text Preprocessing (Python)
- **Design:** Atomic Minimalist / Dark Terminal

## // KEY FEATURES
*   **Dynamic System Status:** Automatic footer status switching (`OPERATIONAL` to `STANDBY / NIGHT_MODE`) based on real-time clock (22:00 threshold).
*   **Auto-Updating Copyright:** Dynamic year rendering using JavaScript.
*   **Refined UI:** Subtle grid background pattern with custom Tailwind CSS integration.
*   **Optimized Structure:** Single-file architecture designed for clarity and educational accessibility.

## // TECH STACK
*   **Frontend:** HTML5, Tailwind CSS
*   **Logic:** JavaScript (ES6+)
*   **Typography:** Space Mono (Google Fonts)

## // ARCHITECTURE DECISION
To maintain simplicity for educational purposes, this project intentionally avoids complex framework overhead. It implements:
- **Vertical Hierarchy:** Logical data flow from identity to external archives.
- **Micro-animations:** `line-grow` CSS transitions for navigation elements.
- **Clean Code:** Transitioned from multi-file structure to a consolidated high-efficiency layout.
